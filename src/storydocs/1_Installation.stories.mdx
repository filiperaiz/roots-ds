# Instalação

Primeiro você precisa instalá-lo usando NPM ou Yarn.

#### Usando Yarn

```sh
$ yarn add @filiperaiz/roots-ds
```

#### Usando NPM

```sh
$ npm i @filiperaiz/roots-ds
```


### Examplo base

```jsx
import React from 'react';

import { ThemeProvider } from 'styled-components';
import { themeTokens, GlobalStyles } from '@filiperaiz/roots-ds';

function App() {
  return (
    <ThemeProvider theme={themeTokens}>
      <GlobalStyles />

       {/* seu código aqui */}
    </ThemeProvider>
  );
}

export default App;
```
