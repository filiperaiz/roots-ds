# Helpers para formatação de valores


## Currency
Esta função retorna o valor formatado para a moeda brasileira ou passando como segundo parâmetro um boleano true, é possivel remover a máscara monetária.

```jsx
import { formatCurrency } from 'roots-design-system'

formatCurrency(value);

formatCurrency(value, true);
```

#
## Data
Esta função retorna o valor formatado para datas.

O segundo parâmetro é opcional. Por padrão é usado o formato **'dd/MM/yyyy'** Para mais detalhes, conferir a documentação [Documentation](https://date-fns.org/v2.15.0/docs/format).

```jsx
import { formatDate } from 'roots-design-system'

formatDate(value, fmt);

formatDate(value, 'dd-MM-yyyy');
```

#
## Percent
Esta função retorna o valor formatado para porcentagem.

```jsx
import { formatPercent } from 'roots-design-system'

formatPercent(value);

```

#
## Hide Email
Esta função retorna o valor ocultando alguns caracteres do email.

```jsx
import { formatEmail } from 'roots-design-system'

formatEmail(value);

```

#
## Remove mask
Esta função retorna o valor formatado sem caracteres especiais.

```jsx
import { removeMask } from 'roots-design-system'

removeMask(value);

```
