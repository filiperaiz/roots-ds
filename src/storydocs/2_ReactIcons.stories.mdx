# Instalação
Para icones usamos o pacote `React Icons library`

Link para documentação: https://react-icons.github.io/icons?name=md


```sh
$ yarn add react-icons

```

```sh
$ npm install react-icons --save
```

### Exemplo base
```jsx
import { IconName } from "react-icons/md"

function App() {
  return (
    <Component theme={theme}>
       <IconName />
    </Component>
  );
}

export default App;
```
